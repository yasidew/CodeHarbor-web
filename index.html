Welcome

Inside CodeHarbor

CodeHarbor is a collaborative research project developed by a group of final-year Software Engineering students at SLIIT. It is designed to simplify code analysis, improve software quality, and promote accessible, developer-friendly tooling using AI technologies.

Automated Code Review & Analysis

Scans code for smells, vulnerabilities, and standard violations offering categorized suggestions for improved compliance and quality

Complexity Reduction through Automated Refactoring

Uses transformer-based analysis to suggest intelligent, structure-preserving code refactorings that reduce code complexity.

Code Complexity Metric

Calculates and visualizes code complexity using extended WCC metrics, helping developers identify hotspots and improve maintainability.

Gamified Accessibility Testing

Automates accessibility testing and engages developers through gamified tasks, making inclusive development more engaging and trackable.

Research Context & Problem Statement.

Modern codebases often grow complex and unmanageable, making it difficult to ensure quality, security, and maintainability.

  • Performed an in-depth literature review analyzing the strengths and limitations of existing static analysis and code review tools.
  • Identified lack of contextual, AI-driven analysis in current tools.
  • Research problem: How can AI enhance automated code review and complexity reduction?

Research Objectives

CodeHarbor aims to enhance code review quality through intelligent automation and visualized complexity metrics.

  • Automate code compliance checking with AI-generated recommendations.
  • Suggest refactorings to reduce structural complexity.
  • Visualize complexity using extended WCC metrics.
  • Improve accessibility testing with gamification.

Research Methodology

The system is built using a hybrid approach that integrates machine learning models, static code analysis, and usability features.

  • Code analysis using AST parsing and LLMs (CodeT5).
  • Training models using annotated code and guideline data.
  • Evaluation through metric-based and feedback-based testing.

Technologies Used

CodeHarbor integrates modern web technologies with AI and data science stacks to ensure a robust and scalable system.

  • Django (Backend Framework)
  • PostgreSQL (Database)
  • HTML, CSS, JS (Frontend)
  • CodeT5 (Transformer-based Code Analysis)

Frequently Asked Questions

Have questions about how our AI-powered code analysis tool works? Check below for answers to the most common queries from developers and teams.

What types of code does the tool support?

Our tool currently supports Python, Java, and JavaScript codebases. We are actively working on expanding support for additional languages such as C#, PHP, and TypeScript.

How does the AI generate recommendations?

The AI engine analyzes your codebase using trained machine learning models and static analysis techniques. It detects complexity issues, coding standard violations, and potential optimizations, then provides context-aware recommendations tailored to your project.

Is my code data stored or shared externally?

No, your code stays secure. All analysis happens locally or within your private project environment. We do not store, share, or use your code outside of the analysis you initiate.

Does the system integrate uploaded coding guidelines?

Yes! The system allows companies to upload their own coding guidelines, which are then incorporated into the analysis pipeline, making recommendations aligned with organization-specific standards.

Can developers track improvement over time?

Yes, the system provides interactive dashboards showing metrics like code quality score improvements and AI suggestion history, helping developers monitor progress across iterations.
